# Runafix Platform

Runafix is a mobile device repair management platform designed with a "technician-first" philosophy, streamlining complex administration so technicians can focus entirely on their craft.

## Key Features

### Repair & Queue Management
- **Smart Queue System**: Automated and organized distribution of service units to technicians.
- **Service Tracking**: Real-time monitoring of every repair stage and technician progress.
- **Simplified Workflow**: Minimal administrative input required from technical staff.

### Service & Output
- **Invoice Printing**: Full support for dot-matrix invoice printing with an integrated print service.
- **After-Sales Management**: Complete tracking for warranties, refunds, and compensation for units damaged during the repair process.

### Technical Excellence
- Developed using the **Autostrada starter kit**, ensuring a robust and enterprise-grade application architecture.
- High-performance Golang backend for low-latency operations.

## Performance

- **Fast Response**: Average API response time < 50ms for smooth technician workflows.
- **Hardware Integration**: Low-latency communication with physical dot-matrix printers.
- **Scalability**: Capable of handling thousands of technicians and service units simultaneously.

## Tech Stack

- **Framework**: Autostrada Starter Kit (Go)
- **Router**: Chi Router
- **Database**: PostgreSQL (Sqlx)
- **Frontend**: React with TypeScript
- **Styling**: TailwindCSS & Shadcn UI
